Cotton

CottonはInput要素の動作を拡張するためのライブラリです

TextBoxは[utf-8, utf-16, sjis]のいずれかで入力されたものとし、必要なByte数を超えないようにすることができます

TextBoxの場合は、設定されたMaxByteプロパティを超えたら、超えた部分の文字を削除するか、Input要素の枠色を強調するかの２通りの動作を設定できます

TextBoxの他のType、NumberやPasswordについては、HTML5の標準の機能で十分かと思いますので、Cottonではこれらに対しては対応しておりません(今後の対応は不明)

CheckBoxはRequire属性を追加できます

CheckBoxのRequire属性は標準のものもありますが、こちらはTextBoxと同じようにChangeイベントが発生した時点でCheckの有無を判定できます

また、FormのSubmitイベントについて、Submitを行なう前にForm下のすべてのCotton要素に対し入力値が正常か否かを判定する機能を用意しました

以上のTextBox,CheckBoxなどのCotton要素に関する使用方法は以下を御覧ください

目次

  • UTF8 - TextをUTF8として扱う
  • UTF16 - TextをUTF16として扱う
  • SJIS - TextをShiftJISとして扱う

TextBox

UTF-8

utf-8で10byteを超えた部分は削除される

<input type="text" class="ctn-text-utf8 ctn-text-leftbyte" ctn-maxbyte="10">

utf-8で10byteを超えた場合は枠が赤くなる

<input type="text" class="ctn-text-utf8 ctn-text-highlight" ctn-maxbyte="10">

UTF-16

utf-16で10byteを超えた部分は削除される

<input type="text" class="ctn-text-utf16 ctn-text-leftbyte" ctn-maxbyte="10">

utf-16で10byteを超えた場合は枠が赤くなる

<input type="text" class="ctn-text-utf16 ctn-text-highlight" ctn-maxbyte="10">

ShiftJIS

sjisで10byteを超えた部分は削除される

<input type="text" class="ctn-text-sjis ctn-text-leftbyte" ctn-maxbyte="10">

sjisで10byteを超えた場合は枠が赤くなる

<input type="text" class="ctn-text-sjis ctn-text-highlight" ctn-maxbyte="10">

Number

入力可能な数値の上限と下限を設定する→標準でできる

<input type="number" max="999" min="1">

Password

パスワードに英大文字、英小文字、数字がそれぞれ１つ以上含まれているか→標準でできる

<input type="password" minlength="8" maxlength="32" autocomplete="off" pattern="(?=.*\d)(?=.*[a-z])(?=.*[A-Z]).{8,}">

パスワードのエラーメッセージを追加することが可能→標準でできる

<input type="password" minlength="8" maxlength="32" autocomplete="off" pattern="(?=.*\d)(?=.*[a-z])(?=.*[A-Z]).{8,}" title="英大文字(A～Z)、英小文字(a～z)、数字(0～9)がそれぞれ１つ以上必要">

CheckBox

Require

チェックがない場合はメッセージを表示する

<input type="checkbox" class="ctn-check-require">
利用規約に同意する

メッセージの内容は指定できる

<input type="checkbox" class="ctn-check-require" ctn-invalid-message="チェックが必要です">
利用規約に同意する

Submit

Check

Submitする前に入力値チェックを挟む

<form action="javascript: alert('all of input is valid!');" class="ctn-form">
   <input type="text" class="ctn-text-utf8 ctn-text-highlight" ctn-maxbyte="10" value="デフォルトで駄目な値">
   <input type="text" class="ctn-text-utf16 ctn-text-highlight" ctn-maxbyte="10" value="デフォルトで駄目な値">
   <input type="text" class="ctn-text-sjis ctn-text-highlight" ctn-maxbyte="10" value="デフォルトで駄目な値">
   <input type="checkbox" class="ctn-check-require" ctn-invalid-message="チェックが必要です"><span>require check!</span>
   <button type="submit">送信</button>
</form>

require check!
